# Image Storage Directory

## Place your photos in the appropriate folders:

- **community/** - Community service, helping people, distribution drives
- **healthcare/** - Medical camps, blood bank, health initiatives  
- **education/** - Universities, schools, scholarships
- **international/** - Foreign visitors, exchange programs
- **sports/** - Swimming pool, gym, fitness facilities
- **disability/** - Prosthetic camps, wheelchairs, rehabilitation
- **certificates/** - 80G certificate, registration documents, awards
- **leadership/** - Presidents, board members, trustees
- **highlights/** - Best 4-6 photos for homepage hero/carousel

## Image Requirements:
- Format: JPG or WebP
- Max width: 1600-2048px
- File size: Under 300KB (optimized)
- Naming: lowercase-with-hyphens-01.jpg

## After adding images:
Update the file paths in `lib/gallery.ts` to match your actual image names.
